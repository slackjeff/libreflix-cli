# LibreFlix Cli

## O que é?
*LibreFlix CLI* é uma versão NÃO oficial do [LibreFlix](https://libreflix.org/) para você assistir seus filmes pelo seu terminal na sua distribuição GNU/Linux.

## Dependência
* youtube-dl >= 2019.06.21
* mplayer    >= 1.2.1-5.3.0
* bash       >= 4.3.48
